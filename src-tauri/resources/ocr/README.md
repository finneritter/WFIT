# OCR models

`text-detection.rten` and `text-recognition.rten` are the pre-trained models for the
[ocrs](https://github.com/robertknight/ocrs) engine, by Robert Knight, downloaded from
the project's canonical model bucket (`ocrs-models.s3-accelerate.amazonaws.com`, same
files the ocrs CLI fetches; also published at
[huggingface.co/robertknight/ocrs](https://huggingface.co/robertknight/ocrs)).
